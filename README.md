demeter
=======

兼容IE6的前端框架
Demeter
注意：文档正在编写中，尚未完成
====


轻量级的、兼容IE6/7/8、响应式的CSS集合




特性
--------

* 响应设计的栅格系统

* 基于typo.css的样式重设、中文友好的排版样式

* 支持`<a>` 和 `<button>`标签的按钮样式.

* 垂直和水平的菜单

* 常见的表格样式

* 菜单

* 表单

* 基于KISSY UI的常用JS组件，下拉菜单、提示框等等



浏览器兼容
---------------------------

demeter在以下浏览器中测试通过:

* IE >= 6
* Firefox, Chrome, Safari最新发行版
* iOS 6.x, 7.x
* Android 4.x
